# 🏪 STORE X SALES MANAGEMENT SYSTEM

## 📋 TỔNG QUAN

Hệ thống quản lý bán hàng hoàn chỉnh cho Store X, được phát triển bằng **C# Windows Forms** và **SQL Server**. 

### Tính năng chính:
- ✅ Quản lý nhân viên (phân quyền:  Admin, Sales, Warehouse)
- ✅ Quản lý sản phẩm (CRUD, hình ảnh, tồn kho)
- ✅ Quản lý khách hàng (lịch sử mua hàng)
- ✅ Bán hàng (tự động trừ tồn kho)
- ✅ Nhập hàng (Purchase Order)
- ✅ Báo cáo doanh thu, lợi nhuận, tồn kho
- ✅ Bảo mật:  Mã hóa mật khẩu, phân quyền chi tiết
- ✅ Audit log:  Ghi lại mọi thao tác

---

## 🛠️ YÊU CẦU HỆ THỐNG

### Phần mềm cần thiết:
1. **Visual Studio 2019** trở lên
2. **SQL Server 2012** trở lên (hoặc SQL Server Express)
3. **.NET Framework 4.5** trở lên
4. **SQL Server Management Studio (SSMS)** - khuyến nghị

### Cấu hình tối thiểu:
- RAM: 4GB (khuyến nghị 8GB)
- Ổ cứng: 500MB trống
- OS: Windows 7/8/10/11
